# Supply Chain Game

**Live Demo:** [supplychaingame.net](https://supplychaingame.net)

A **web application** that simulates the **Beer Game (MIT Supply Chain Game)**, developed for use by the **System Dynamics Society** in guest lectures for **business students**.  

The simulation models supply chain dynamics and decision-making across multiple roles (**Retailer**, **Wholesaler**, **Distributor**, and **Manufacturer**) to demonstrate the effects of information delays, order variability, and system feedback loops.

## Features

- Interactive real-time gameplay for multiple participants  
- Role-based dashboards with inventory and order tracking  
- Visualizations of supply chain performance and system behavior  
- Admin tools for facilitators to create, monitor, and manage games  

## Tech Stack

- **Frontend:** React, TypeScript, Vite  
- **Backend:** Node.js, Express, Prisma, PostgreSQL  
- **WebSocket Communication:** Socket.IO  
- **Hosting:** AWS EC2
